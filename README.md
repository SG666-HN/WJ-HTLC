# WJ-HTLC
本项目是论文《Attack and Defense Methods for Miner Extractable Value in Blockchain
Transactions》的实验代码，开发了一个基于Solidity语言开发的抗MEV攻击的新型跨链原子双向交易合约（WJ-HTLC），合约通过时间锁机制以及多重惩罚机制，既能防范传统/新型MEV攻击，又提高了交易效率以及成功率。  
This project is based on the paper "Attack and Defense Methods for Miner Extractable Value in Blockchain."
The experimental code for "Transactions" developed a novel cross-chain atomic two-way transaction contract (WJ-HTLC) based on the Solidity language to resist MEV attacks. The contract employs a time-lock mechanism and a multi-penalty mechanism, effectively defending against both traditional and novel MEV attacks while improving transaction efficiency and success rates.
